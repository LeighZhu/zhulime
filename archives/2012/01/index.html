那仅存了科幻，只因童话已不再

2012 Jan 26

被惊醒了数次，还不知道，原来早已寒假，曾每年期待的除夕已过去好多天。。

起身，却回忆起大一，那时怎会想到，原来也是无忧无虑。

梦中一切，无非是得与失，于是也给自己取了个名字叫噩梦小王子。

爱做梦，是人类和某些生物共同拥有的一个美妙的玩具，无论平复贵贱，高矮胖瘦，闭上眼，任你飞禽走兽。

看过盗梦空间，其实剧情并没多少匪夷所思和需要多少逻辑，其实很多人都有过这些经历。

人说梦是自己造的。

为何不给自己全部造成美梦呢。

甚至梦里又做了梦，梦里的梦，梦到一点好的的东西，上一层的梦里，居然意识到，这太好了，不可能发生的，于是去改掉了，等到醒来，才发现原来作出“改掉”这一行为的，也是在梦里罢了。

盗梦空间中，人们用一个自定的图腾来分辨是否自己在梦里，我怎么可能有那手准备，于是，在梦里，反复地思考一些周围的现象，是否符合逻辑，按照惯例，不符合逻辑，便自己会吓醒，然后，这次没有醒，反而又做了一个梦。。

噢，原来这就叫梦中梦。

等到醒来，我反复核对，我已经考完试，通过了，放假在家了，才慢慢回过神，原来，所谓还有一场考试在明天，只是一场梦；原来……

其实冬天已经没了影，望出窗外，除了杨柳，一切都是碧绿的，连地上，都是绿油油的青草，若不是身上还穿着冬日大衣，怎敢想象，腊月刚过才四天。

转眼间，已然大三。

来得太快，成长太慢。

不成熟的思想，总是想要寻找刺激，懵懂青春，只是空空燃烧。

待到时机即成，却童话不再。

我爱折腾事物，却不习惯折腾的生活。

我也想去飞翔，在蓝天和碧海之间。

而，现实，总会和想象的差那么一点。

那不是缺陷，只是不再在梦中。

年度总结和年度方向

2012 Jan 23

大家都在元旦弄总结，我喜欢在春节弄，哈哈。

回顾过去一年，其实该说是什么都没做的一年。没有看几本书，没有看多少电影，没有听完几张音乐，好久没买 Hit 了。

简单列一下去年那些事：

1. 明确了我准备做前端的方向。

2. 拥有了两本书，《CSS 权威指南》和《Javascript 权威指南》。

3. 顺利通过了两个学期的课程。

4. 参加了七月份淘宝技术嘉年华暨 D2 前端论坛。

5. 写了 favbox 主题。

6. 接下了 学院新网站 的任务（团队）。

7. 开始系统学习 js 。

8. 读过的书：《秩序之美: 网页中的网格设计》、《Web 标准实践:Web 设计的整体方法:a holistic approach to Web design》、《吃掉那只青蛙: 拒绝穷忙，把时间留给最重要的事:21 great ways to stop procrastinating and get more done》、《编写高质量代码.1,Web 前端开发修炼之道》、《JavaScript 学习指南》、《大巧不工:Web 前端设计修炼之道》、《JavaScript DOM 编程艺术》、《Facebook: 关于性、金钱、天才和背叛 》、《 CSS 精粹》、《设计网事: 互联网产品设计实践》、《 精通 CSS 高级 Web 标准解决方案》

9. 保持了和 Zoe 的关系。

欠缺的：

效率太低，看书周期太长，GTD 学了点皮毛，还是不能顺利得用好。

缺乏大的成果，这也怪去年没什么追求，就追求了不挂科。

等等。

==========================

好吧，其实总结不重要的，还是给自己定一下龙年的方向比较重要，大家喜欢弄年度计划，不过我更喜欢叫做方向，计划什么的，月度吧。

1. 完成 javascript 核心部分的系统学习，并且学习现成框架诸如 jQuery，学有余力，则 canvas 绘图和 HTML 5。

2. 入门 android apps 的开发（选了课），若有能力，开发一个简单的 app。

3. 学期结束前，找到暑期实习或长期实习机会，现在目标为 alipay。

4. 顺利完成第六学期的学分和除毕业设计、论文等硬性学分外的事情。

5. 通过大外六级考试，若有余力，则学法语。

6. 继续维持和 Zoe 的关系。

附注：没列出的有硬性任务，如院网之类的，以及习惯之类的，诸如，读书（购入 kindle touch），切实实践 GTD，尝试严格的月度计划，找到工作（球饭碗 :cry: ），等等。

好吧，其实挺潦草的，因为这时间跨度挺大的，不过其实也简单，龙年最大的目标就是找工作。

内容绝对扯淡

2012 Jan 18

昨日，难得出来个“艳阳天”，大中午的，在等待热水烧开的时间里，打开了久违的电视机，然后忽然意识到，对哦，这都数字电视了 XXDTV 了。

翻来覆去，电视还是和以前差不多，每周星期二检修机器，然后放个色块给大家看的习惯还是存在，少数电视台貌似不实行这个习惯了。

看了半集电视剧，然后换台时，被一个电视购物深深的吸引住了。

299！ 只要 299！黄金镶边手机带回家，再送你价值 299 的黄金手表（注：我不知道是不是听错了。。。），等等，还送 XX 公司自主研发的 XX 软件一套，然后画面中出现一个芯片的图形。。汗。然后介绍了有了这个芯片的，软件的东西后，可以免费通话，免费短信，不花钱！！之后是介绍四核处理器，高速上网，比电脑还快 = =！！惭愧啊，学通信的孩纸，至今还没发现在大陆，移动上网速度可以比电脑还快的方法。。

然后 balabala 一大串，大家懂的。

循环了两遍后，我在电视机画面中某个不起眼的地方，发现了一行小字——手续费 59. OMG，原来天机在此。他该是这么赚钱的，很多不明真相的观众，大气电话订购，然后接通后，给你磨叽几句，然后跟你说，货已经卖完了，但是接通了，手续费嘛～～～咩哈哈哈哈～～～～

然后发现这热线电话居然是 400 的，自从 800 变成行骗好工具后，400 也载了啊。。。哎

苦了国内那么多年迈的，观察力不太好，不能发现那些小字的，不懂软件和芯片的，不懂通信的，不懂电子的，不懂交换技术，更不懂监管的长辈们。

安安稳稳看个电视容易么！
