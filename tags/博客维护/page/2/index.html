新主题

2010 Aug 21

WordPress 更新到 3.0 后，内置的主题换成了 twentyten ，拜读了一下它的 functions.php，顿时被它迷倒了，于是开始做新主题。

于是就有了这个主题 ：Silu Beta

采用了 twentyten 为基本框架，改动得还算多，目前主要增加或修改的（twentyten 相比）有：

评论区 Ajax 化，即发表评论无需刷新即可看到；头部、主题、页尾，使用 CSS 阴影、圆角效果；头部使用 RGBA ，做出半透明毛玻璃效果；增加 Most Actived Friends 即读者墙功能；侧边栏 twitter 免翻篱笆调用……

其他的就是各种小幅修改即插件实现的效果了。

据首批测试者反馈表示，看久了容易眼花，好吧，标题文本阴影可能用太重了，另外应该是背景图惹的祸啦~

另外要注意的是，新主题在 ie 各种版本、opera 下面极其朴素，firefox 3.6 以下版本可能也会缺失部分效果，所以建议使用 firefox 3.6+ ，或者使用 Google Chrome 、Safari 等 webkit 内核的浏览器来获得最佳体验~
