火狐中国版的特色扩展

2011 Mar 18

惊闻 Firefox 4 将于本月 22 号发布下载，作为 Firefox 1.x 开始的用户来说，还是小小兴奋了下，虽然早已在 写完这篇 Firefox 扩展 文章后乐不思蜀在隔壁 Chromium（Chrome/ 谷歌浏览器）的暧昧下。

当初选择 Chrome 的原因，也是 Firefox 用户频频抱怨的原因：启动速度慢，折腾几个扩展，打开就要用分钟计算。然后在 Chrome 的 V8 引擎左敲右击下，已经对 Firefox 的各种速度无语，加上 Chrome 强大的云同步，哇塞～～～

之后，Chrome 便成了主浏览器，Firefox 只在开发时候打开来。

去年底，Mozilla 终于在中国这个残暴浏览器战场下手了，上线了 firefox.com.cn 哇塞，这设计，这体验。。。
