好久不见！

2010 Nov 13

前几天很邪门地上火了，嘴痛到吃饭及说话的不能，而后一睡觉，喉咙烧痛得像喝了烈酒之后。

很不幸，迎来了重感冒。记得那天是一节课都没去上，下午去参加了什么二分院什么新闻中心什么写稿培训，走到十二教旁边还晕了一下，差点以为我能飞了。

关掉了在人人网的情侣空间，因为它本就不该出现。有些事，别人即使看到了，也是过目即忘，不如藏起来，等到多少年后，再拿出来，给爱听的人讲讲曾经的往事。

晚上去拿了网页设计大赛的设计稿，很特殊的设计稿，手绘的！！回到寝室草草用 HTML 和 CSS 画了基本框架，然后就撑不住了，直接睡了去。

星期四上午上了三节课也回了寝室，下午的体育课也没去，直接在寝室休息掉了。顺便把计算机学院的网站上的文字及连接写了一遍。晚上实验毕，给 太阳系的小鬼 举行了庆生活动，这个和 苍井空 老师同生日的朋友，不知道还有多少次“太阳系的小鬼”可以去叫。

很多人不知道为什么我要把博客的名字定为“Leigh:// 斯路”，那么，今天我就来解释一下。

仅仅例行

2010 Oct 16

已经进行了一个礼拜的真正的大二学习了，基本每天就是在各种上课中度过了。

周末，习惯性地时刻，就被自己叫醒了。

最近，还是不怎么适应，大量的东西下来，应接不暇。

对于之前收集的知识，遍布在 Google Reader 、Twitter 的各种星标下，以及浏览器书签工具栏的 Temp 文件夹中，极度散乱，在用的时候，很囧了，找不到收藏的，搜不到以前看过的，很囧，于是准备做一个 Personal Wiki ，这该是对我硬盘中的 Library 目录的一个补充.

参考了很多文章，选了 moinmoin 这个 wiki 程序，基于 Python ，不采用数据库的存储方式，折腾了好几天，才发现桌面版并不需要怎么折腾，于是就爱使用。起初很不习惯 wiki 语法，很烦很繁。后来看到很方便地目录组织，顿时喜欢上了 wiki ，本来还打算转到主流的老牌的 MediaWiki ，怨念的是 xampp 现有的非 beta 版刚好是不兼容的 PHP 版本，于是搞了 beta 版，结果死活搞不定 ipv6 造成的数据库不能连接的错误，果断弃之，回到 moinmoin 。

这是这几天折腾地最多的一件事了。

这样一折腾的后果就是，我把原来的本地环境删了，结果 SiLu Beta 这个博客主题的新版也没有了，所以又从之前的备份档重新改了一遍……
