新主题 favbox

2011 Feb 1

这个主题原本是为 Zoe 写的，写完了，我也忍不住换上去了～～

Favbox 这个名字是乱取的，favorite box ？ 我也不知道，姑且是吧。

废话不多说，下面介绍主题

1. 运用 HTML 5、 CSS 3、 jQuery 1.4.2 创作。（故，原则上支持 IE 9、Firefox 3.6.X、Chrome 等标准浏览器，用了 javascript 使的 IE 9 以下版本也支持 html，但配合 jQuey 、CSS 3 实现的头像圆角效果，IE 就。。。）

2. 采用 wordpress 的新函数，比如 wp_nav_menu 。

3. 符合 W3C 的 HTML 标准，CSS 除各种无聊蛋疼的私有属性之类的外，也符合标准。

然后是主题功能特点

1. 集成 最新评论、自定义表情路径、按月归档存档页、page-nav 这些原本用插件实现的功能。

2. 自带留言板模板，实现倒序排列评论及将“楼层数”改为“高度 / 深度米数”。

3.ajax 评论提交，提交评论后无需刷新，并且增加快捷键 “ctrl+enter”。

4. 评论部分，点击 “Reply”链接，自动在评论框加入 @XXX 链接。

5. 存档页为按月存档，展开效果使用 jQuery ，下边则是彩色的标签云。

6. 鉴于 Clicki 网站打不开，将侧边栏访问追踪换成 feedjit 。

7. 侧边栏实现读者墙。

到这，主题差不多没啥好介绍的了，今后还会一步一步做调整，请大家期待～～

PS：主题开发借鉴了 zwwooooo 、 bigCat 等很多大牛的 code ，在此表示感谢。
