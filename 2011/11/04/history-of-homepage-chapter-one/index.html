织网历史-Chapter One

2011 Nov 4

今天在整理东西的时候，发现了几个曾经胡乱备份的文件夹，在里面找到了曾经做过个人网站首页的文件。。居然还在的。

 

回顾一下这段历史，其实大概是要从当年 cn 米一元一个坑爹白菜价开始说起，从我现在博客上的 ISP 备案号来看，我买米应该是 08 年的时候，一个 cn 米。

说道买米的动机，其实还有一段历史，

其时间大致可以追溯到 2006 年中考结束，因为在中考之间休息的三天，我注册了一个所谓个人主页，当时是在 51.com ，清楚记得“51 我的家”什么的，中考后，便开始弄，发现特别有意思，后来知道，这原来叫 Web 2.0 。

后来，玩了几天，我发现 51 并不好玩了，虽然它可以发文章，发图片什么的，应有尽有，也可以有属于自己的 2 级还是 3 级域名。

然后知道了博客这个东西，我的第一个博客应该算是在网易的，当年网易发布博客这个产品的时候，好象是叫“网易部落阁”什么的，很山寨，因为有 163 邮箱，开通后，贴了篇文章就不知道了。

然后 2006 年，大家懂的，博客时代来临啦。

玩了好多 BSP 发现也没有意思，太局限了，尼玛只能换个图片，然后也是 06 年的样子，百度发布了它的博客产品——百度空间，然后它支持一个高级的东西，叫做 CSS 。。这是神马玩意，一窍不通，研究“百度空间”贴吧中各位大牛分享的代码，犹记得当年百度空间还可以 @import 还可以 javasc+ript ，甚至还可以 XSS 跨站。。几乎可以完全按照自己的意愿来布局。

这尼玛太强大了，于是花了一个晚上去自学 CSS ，后来终于也成为一头小牛，在贴吧里回答各种 CSS 问题。

记得当年贴吧里刷贴很牛逼的一个人叫“胡戈戈”，后来他先一步投奔了 wordpress，带着他的 kisshi.com 飞黄腾达去了。然后我作为一个准高中生，默默无闻地写化学方程式去了。

高一的时候，最喜欢周末跑大老远去市区最大的那个新华书店看书了，直奔计算机相关的那些书架，什么 FrontPage 入门 ，Dreamwave 入门到精通乱七八糟的，在那蹲了几个星期后，忍不住买了一本，洪恩教育的 Dreamwave 的，因为看其他书中写道 FrontPage 是垃圾，圾。。

接着，花了好几节数学课和语文课，把它看完了，很幸运，也很感谢的是，那本书并没有只讲如何“所见即所得”，而是讲了一些代码。

然后，幸好学校的图书馆居然也有几本讲 HTML 的书，纯代码的那种。。。那个兴奋，然后又看完。最不可思议的是，我居然当年没有学 table 布局。。。直接 DIV 。。。（大牛：你丫 DIV 和 table 一样用）

去网吧以及图书馆各种上网搜索各种免费空间，写出了当年第一个网页，可惜的是，好像找不到了。

然后，从某本书中知道有动态页面语言这个东西，可以让新浪几万个网页不用手动写。。写。

手拿 HTML 和 CSS 的我，当然耐不住寂寞，经不起 月光博客 用的 Z-blog 的诱惑，蠢蠢欲动，也想有个独立博客。时逢 2008 年白菜 cn 米，买了一个，又不知从哪买了第一个虚拟主机，应该是 Z-blog 官方。搭起了博客，然后因为要写的化学方程式太多了而没再去管。

哎呀呀。。太长了，这是第一章～
