如何在 Web 上学习

2011 Nov 26

了解自己的学习方式

1. 视觉和书籍学习者：比较善于使用眼睛获取知识，“看”便能得到最好的学习效率。

  • 将听到的信息变换为图片或文本。
  • 将思路变为思维导图
  • 做笔记
  • 在安静无声的环境学习

2. 声音学习者：比较善于听取信息，通过讨论交流能得到很多信息的。

  • 积极参与讨论
  • 对课堂或讲座采取录音或视频记录
  • 大声朗读
  • 用口头表达思路，并使用故事类比，使自己记住

3. 运动学习者：很难静止很长时间，希望周围人都能停下讨论，谈话的时候会因手的姿势分散注意力的人，如果某人展示一个做事方法，很容易重复起来。

  • 阅读之前先略读
  • 站着阅读或学习，同时做些运动，如嚼口香糖。
  • 学习的时候，手上做些事情（转笔，折纸）
  • 来回移动的时候，暂停学习

找到自己最能接受的类型资源，从 Web 上找资源的时候，可以有所取舍。（图书，音频博客，视频）

搜索

  • 确保每个关键词都起作用
  • 使用在寻找的页面上可能用到的词
  • 使用描述性词，并使用尽可能少的词
  • 短语使用引号。
  • 使用搜索操作符
  • 过滤结果：1）阅读链接描述   2）检查 URL，是否权威   3）查看页面缩略图（Google） 4）检查作者可信度 5）时效性   6）不要放弃一些不美观的页面

引用资源工具：http://easybib.com

 

做笔记

1. 保存到云中：

  • 博客（BSP，或者 Wordpress 自建博客）
  • 添加到书签网站（Delicious.com 或 Gnolia.com ），Leigh 注：大陆用户不推荐这两个网站。
  • 使用在线笔记本（Evernote，Wiz）

    • 2. 传统笔记：使用最习惯的纸笔方式记录，目的只有一个——用的时候找得到。

    3. 文本档案：保存成普通文档，存在硬盘里，如果需要在网络上访问，可使用 Google Docs。

    Leigh 注：推荐使用 1 ，以及使用 Evernote，大陆及不使用 Mac 的中文用户，可以使用 Wiz，比 Evernote 更适合国人习惯。

     

    保持知识更新

    1.RSS 跟踪： Google Reader 等

    2. 播客：使用 iTunes 等软件订阅，适合于音频学习者和运动学习者。

    3.Email 订阅：适合经常使用 Email 的人，缺点是会有风险收到大量垃圾或无用信息的邮件。

     

    关注任务

    1. 生产力系统（效率）

    • Geting Things Done
    • 43 Folders
    • Zen to Done (Simple version of GTD)
    • 了解自己的工作习惯
    • 了解自己的生理周期
    • 理解生生产力系统的原理，投入使用
    • 以简单为目标

    2. 使用日历进行长久任务管理：Google Claendar

     

    他人处获得答案

    Twitter， Facebook 等，通过发帖求助而获得回复等。

     

    提供回答

    1. 做一个良好的在线市民：建立自己的威信度，可信度，注意礼节，任何不良的言语都会对自己以后产生影响。

    2. 确定自己的身份，将自己的博客主题，或者在其他网站所发表的帖子，都是自己所处的行业相关知识。

    3. 礼仪：

    • 参与讨论之前，先花时间观察已经产生的讨论
    • 提出观点时，知道自己在说什么，说有意义的内容
    • 开放的分享你的知识和经验
    • 链接补充信息
    • 留下你的信息资源——思路，故事，图像等
    • 留下有帮助的评论，读一遍自己的留言，确保表达明确
    • 不要说废话或自言自语
    • 对别人的错误宽容，注意告知其真相的语气

    4. 反馈：

    • 通过博客评论，将自己的评论写成博客，参与讨论。
    • 在论坛和邮件列表中分享自己的经验和资源等
    • 通过社会媒体，Twitter，Facebook 提供对他人有用的帮助。
    • 建立自己的 Web 站点，分享资源。

     

    协作

    与他人一起工作一起交流。

     

    灵感收集

    建立一个百宝箱文件，收集自己随时想到的创意，灵感，短语，词句等等所有可以保存的，可以刺激想象力的东西，方便以后在创作，设计，工作时查阅。那时，你会很庆幸自己曾经保存过那么多能激发灵感的东西。
