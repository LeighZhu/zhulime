大学物语-第一弹

2009 Oct 3

先别激动，先做好准备，本文内容花了我一个多小时整理，内容丰富，但关系有点复杂，需仔细看方能看懂……若你现在很忙，请找空余时间再看！

这些事是我进入学校后二十多天内发生的，全部属实，为保护隐私，均以代号相称，当无聊剧本看。无恶意，搞搞别生气。 ：） 好了，可以激动了 ！

前奏： 某拿起手机，随手拨一个号码，正好是 C 君，没接通即挂断，另一领会意思，除 C 外的均准备开刷…… [J 为虚拟人物，女性，由三人驱动……C 即某君，缺乏生活阅历的某君……Y 为驱动组三人之一]

J：你是 CY 吗？

C：刚刚接电话的是我同学 你是？

J: 呜呜，我刚才不是跟你说了吗？JCY 工商管理的。

C: 那啥事啊，找我啥事啊？

J: 寝室女的很吵，很多话想对你说，能出来见见吗？

C: 先发短信

J: 你还是不是男人啊！我不是那么随便的人。

[C 拨通号码，接通，驱动组三人之一接到。那人是重要角色，姑且称作 Y]

C: 你的声音好怪。

J: 楼下大爷真吵，本来人家就感冒了，你不介意吧？所以我说信息嘛

J: 呜呜，你欺负我。不管怎样，C，我想见你，你也想见见我吧。

C: 我们几时见过 哪节课 你坐在我哪个方向

J: 什么啊，真的假的啊。笑死我了。心理课上就在你前面呀，不和你说了，感冒不舒服。

C: 我有老婆了

J: 你这人真有趣。我又不叫你做我的……那天啊见你跨桌子就觉得你很有意思了，呵呵。果然呐……

C: 我这个人很没劲的 我很无聊的 讲句老实话 你吓了我一跳 有缘分的话自然会见 明天就算了吧

J: 我们可以先聊聊天吗？我先睡觉了。

C: 短信先聊聊。我确实有女朋友，在浙江金融职业技术学院，明天算了哦，要好好学习哦，晚安。

J: 浙江金融职业技术学院？那是什么学校啊？你老提你女朋友干什么，真是搞笑哦

[……1 天多的时间，C 君对 J 念念不忘，有事没事就扯出来说……]

J: 有人说你每天提到我…

C: 我同寝室说你爱上我了，昨天下午在 XX 楼一下午，吓了我一跳。

J: 你为什么不去啊？你知不知道这样放人家鸽子很过分啊！

C: 我三次说不去的，难道你没收到短信。 C: 你别吓我啊，我好怕怕啊。 C: 有人把你说成结婚狂似的，虾死偶了（注：原文如此）

J: 你很搞笑！

C: 他们描述的你和我想的不一样。 C: 不说了，哥哥写作业了。

J: 你太让我失望了，我觉得你根本不算个男人！你以为你是谁啊！自我感觉不要太好！我想认识你是看得起你，既然你这么不给面子我也不想和你说什么了。你这样的男人我鄙视，再见！

C:……

[J 拨打 C，只响一声]

J: 按错，别误会。我把你删了，以后你会再收到短信了，你自己好自为之，不再见了！！

注解： 1. 关于 C，是一个离开父母后，洗衣粉都不会用；洗澡不会洗的人。 2. 关于 C 口中的女朋友：他的官方解释——因为以前和她话比较多，然后那个女的没有男朋友，所以 C 说自己是她男朋友…… 3. 驱动组之一即接通的电话的是 C 的初中同学，居然没听出来…… 4.C 君绝非无厘头搞笑，而是真的很悲哀地无厘头地生活着……

附赠小故事：

1. 注解 2 的插曲大致对话，W 代表驱动组：

W: 你怎么说她是你女朋友？ C: 因为以前和她话比较多。 W: 话多就是了啊，那我不是有后宫群了。 C: 因为她没有男朋友 W:&……%&

2. 某日上课，临近吃饭，C 发短信给 Y：“十一点之前要半吃饭。”[绝对原文]

3. 某日下课，路上桂花飘香，C 君：“这个桂花的香气怎么有点臭。”

4. 某日，C 君跑到一个人面前，问道：“我能不能问你一个问题？”然后不说话了，拿了一本书走了。（问题是：“能不能借我一本书。”）

5.Y 喜欢同班某女 R，但 R 说已经有喜欢的人…… 某日，C 君说道：“Y 没有女朋友” W: 你怎么又知道…… 我：我知道的，根据他的定理：C 虽然喜欢 R，也经常聊天，但 R 有喜欢的人，而且 R 喜欢的人没女朋友，所以 R 是有男朋友的，所以 R 不可能成为 C 的女朋友，所以 C 没有女朋友…… Ps：这是本银推导的 CY 第一定理！

额弥陀佛……
