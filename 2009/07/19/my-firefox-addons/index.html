我的Firefox扩展

2009 Jul 19

一直用 Firefox 作为主要浏览器，因为它有强大的扩展，所以，就算 Firefox 资源占用高，就算它的启动时间差不多等于系统启动，我还是离不开它。

下面的列表时是由 MR Tech Toolkit 这个扩展生成的。[这是我的精简版扩展列表]

生成： Sun Jul 19 2009 10:06:10 GMT+0800 User Agent: Mozilla/5.0 (Windows; U; Windows NT 5.1; zh-CN; rv:1.9.1.1) Gecko/20090715 Firefox/3.5.1 Build ID: 20090715094852

启用扩展： [11]

  • AutoProxy 0.2.1+.2009032705 ：在大陆上网，总是需要切换代理，而这个扩展可以帮你自动切换，而省去了手动进入选项的麻烦过程。
  • Delicious Bookmarks 2.1.041：Delicious 是雅虎旗下的著名在线收藏网站，它拥有导入导出功能，所以我选择它，它基本代替了我的 Firefox 收藏夹，另外，对于 Firefox 3.0 + 版本，它还可以集成到 地址栏的搜索结果里。
  • Easy DragToGo 1.1.2.4：很小巧的拖拽扩展，上网浏览更加方便了
  • FireGestures 1.5.1：鼠标手势扩展，有很多手势可以定义。
  • Greasemonkey 0.8.20090123.1：强大的脚本扩展，类似于 bookmarklet ，但又强大很多，使用脚本可以实现 自动转换 http 为 https 、同时登陆多个 Gmail 帐号等。
  • IE Tab 1.5.20090525：对于某些 IE only 的钉子户网站，以及国内的网银，可以用这个来解决。
  • MR Tech Toolkit 6.0.3.4：此列表生成工具，目前只用它这一个功能。
  • NoScript 1.9.5：安全工具，可以对不信任网站的 Js ，iframe 框架等移除，还有 clearclick 保护等等，基本避免了普通的网页 挂马。
  • Tab Mix Lite CE 3.0.5：原本使用 TMP 的，但它跟不上 Firefox 版本更新，所以换用这个，功能也差不多够用了。
  • Tiny Menu 1.4.10：把 Firefox 的菜单栏集成到一个按钮，这样就可以把地址栏、搜索栏一道顶上，多出了一个工具条的空间，增加可浏览面积。

  • TwitterFox 1.8.3：Twitter 在 Firefox 的扩展，支持自动登录、可显示 回复，私人信息。支持 RT 。 已安装主题： [1]

  • Default 已安装插件： (6)

  • Adobe Acrobat

  • IE Tab Plug-in
  • Microsoft? DRM
  • Mozilla Default Plug-in
  • Shockwave Flash
  • Windows Media Player Plug-in Dynamic Link Library 后续：以前我装的扩展有 20+，但部分只是用来改变外观，或增加某些特效，所以删除了，另外有些扩展岁 Firefox 升级也用不着了，Firefox 已经原生集成了某些功能。

另外，我没有安装流行的Adblock Plus，因为 Firefox+Noscript 已经屏蔽了一些讨厌的弹出窗口，其他的不妨碍浏览的广告，就不必赶尽杀绝了，何况 abp 有时候还会误屏蔽正常内容。

这篇文章用来引导新手，也作为我自己的扩展备份，Firefox 老用户估计学不到什么有用的了。
