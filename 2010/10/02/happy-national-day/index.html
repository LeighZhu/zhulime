国庆节叨叨叨

2010 Oct 2

今天，好像是国庆第二天了，也是长假第一天在家，真的很久没有更新有价值的博客文章了，主要是因为，这段时间，没有在做什么好玩的事，无非就是学习和学习和学习。

昨天提交了下前天晚上，红色家园（学校社团）设计部招新的作业，应该算是前天晚上就完成的。寝室里的人都在玩通宵，于是我也没得睡，干脆写起来。

这差不多是我第一次对着一份 PSD 设计稿，从零开始写的网页，代码量不是很大，html 文件 62 行，CSS 98 行。用 editplus 一个一个码了出来，花了两个小时差不多。

在欢快的“ Code is poetry”进行时，从中发现了自己的软肋，因为在 Chrome 下开发，所以，各种 div 各种 float ，然后 ie 下就悲剧了，差不多有一半时间是在兼容 ie 上花掉的，后来终于把 ie 5～8 都兼容了，我真是天才哇～

今天给博客换了两张图，配合国庆，本来只是随便换换的，结果是 google 给我了这么一个 banner 昂，不用白不用。放上截图留念。

之后也发现了现在主题的未完善处，以及新的审美差异，当初草草写完主题就放上来了，一直没时间修缮。

然后，难得开了 Firefox 看了下，才发现 CSS 3 的 border-radius 属性，碰到图片时候，果真就悲剧了，再加上 box-shadow ，就更囧了，如下图：

可惜清晰地看到，图片的角尖尖的，然后阴影的角圆圆的……

顿时想到了曾经看过的一本书中写的，超越 CSS ，各种浏览器，各种显示方案～～～

扯多了，回到正题。

正如开头所说，因为很久没更新含金量的文章了，所以，下次有质量的文章应该在大大折腾 Ubuntu 之后，也就是 10 月 10 日 U 10.10 发布之后吧，大概～～
