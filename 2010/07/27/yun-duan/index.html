“云端”那点事

2010 Jul 27

这两天，遇到了几件比较郁闷的事，其中“云端”惹的祸，算是解决了。

其实“云端”这个软件挺不错的，自从他前年获得“中国优秀软件创新大赛金奖后”我就开始用上了。

简单的介绍，“云端”提供一种新式软件安装方式，你不需要去软件的官网下载，而是可以直接在云端平台搜索，下载，安装，所有软件都可以轻松的在两步内安装完毕，这到有点像 Linux 平台的部分软件平台。

此外，云端的软件安装方式，是通过虚拟映射一个目录来达成安装，实际软件所有文件，处在一个用户设置的缓存文件夹，这样的好处自然有很多：

1、不会对系统造成任何影响，因为软件所有产生的文件，都只保存在缓存文件夹，其中甚至包括注册表信息。

2、系统重装后，可以瞬间恢复，（保留缓存文件夹），原理同上了，除此之外，云端本身也是绿色软件，所以在重装之前做好简单的拷贝备份即可。

3、满足软件试用狂，在你想试用一款新软件，又怕它对系统造成不良影响的时候，可以把它安装到云端，原理也同上了。

其他众多功能以及介绍，还是看官方的 说明书。

那么，云端在这几天给我惹的祸是……

云端自从 beta 到了 1.0 之后，有了一个可选的服务项，启动该服务后，可以做到不启动云端软件平台时使用安装在云端里面的软件，但是，不幸的是，这个强大的功能目前还非常不稳定，至少在我的 Windows 7 环境下，一直在闹腾。

1、右键新建菜单变空，这个问题好多网友也遇到了，传统的方法是进一次“安全模式”，不过这治标不治本了。

2、影响 Chrome 的 Switchy！ 扩展正常工作，这个应该是云端的服务影响了 IE 的代理设置，而 Chrome 目前是调用 IE 的代理设置。这个问题困扰了我很久，在我查证右键新建菜单问题后，以及反复思考，问题第一次出现是在我打开云端的服务之后，我手动关掉了服务，测试了下，确实 Switchy！ 可以工作了。

其他的问题暂时没有发现，我也已经关掉了服务，国内软件界难得有这么优秀的软件，作为资格绝对老的用户，我整理了下上述发现的 Bug ，已经提交给官方。

应该在下一版本可以得到解决了。

//update 18：00：已经收到开发者的回复邮件了：

“右键菜单的问题 一般是软件造成的 隐藏你激活的软件就没问题了 找出导致问题的软件 修复一下即可解决，代理的问题已经在更改中。”

期待新版本的到来吧
