三天流水

2010 Sep 27

这学期第一次认真听课，因为有人给的激励。感觉还不错，老师进度挺慢……

另外，早上一二节的 现代经济管理基础 课，还是大部分人的补觉课，两节课之后，走出教室，听到的最多一句是，“真好睡啊～”

下午去补考了高数，没进考场时，有人说，这个监考变态来……进去了，拿到试卷，顿时，我以为我穿越了……

回到寝室，看了下书桌上从图书馆借来的书：《CSS Web 设计高级教程》、《CSS+DIV 网页样式与布局案例指导》、《网页配色黄金罗盘》。怎么看都只有讲前端的书，本来是去借一本关于 CCNA 考试的辅导书的，结果只发现了一本很破很破的，作为一个完美主义的偏执狂，绝对不会捧着那么一本书看得高兴的，我宁愿看电子版。（PS：上面提到的书，在豆瓣上，有两本是我传的封面，怎么能容忍没有封面呢……）

还有一本是叫《决战 SNS》，这本书是貌似在九楼社会科学找到的，不过，翻了下，发现大部分是讲 SNS 类网站，怎样去设计人机交互的，哎哎，还是前端啊。严格来讲，放错书架了吧，书内页 CIP 数据给了两个分类，一个是 C912.3 ，一个是 TP393.4 ，个人认为放 TP393 的书架更好。

这本书呢，很新，估计没人看过，我翻到了 70 多页之后，终于来点有用的信息了，还是些前端的内容。就当复习下 CSS ，有些东西，还是可以说是精华的，争取在国庆之前消化掉。

晚饭后，去了曾经室友的寝室，趁着 Google 过生日，给他介绍了很多 Google 的玩意儿。

此外，在打了多个 10000 电信客服后，我成功地成为了一个 “有着微软的 Windows 7，用着微软的 Ubuntu ，还有着微软的 Macintosh ，以及不肯使用微软的 Windows XP 的人” ，好吧，客服姐姐们，你们赢了！终于找到客服经理，终于是个知道 Linux 不是微软家的了。然后，貌似我的闪讯帐号准备免客户端处理了。

这几天邮件些多了，文章风格变了。哎，又是一篇没有营养的文章，只好叫它 流水帐。

#nowplaying: 五月天 -《 雌雄同体》
